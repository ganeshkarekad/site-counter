Site Tracker Extension Test Page

Use this page to test the Site Tracker Chrome Extension functionality.

📊 What This Extension Tracks

  • Visit Count: Number of times you navigate to a domain (including current visit)
  • Time Spent: Total time spent on each domain (automatically pauses when tab is inactive)
  • Daily/Weekly/Monthly Stats: Filter visits by time period in the popup
  • Smart Navigation Detection: Only counts real user navigation, not background requests

✅ Expected Behavior

  • First Visit: Popup shows "1 visit today (including current)"
  • Revisit: Popup shows increased count only after 5-minute cooldown
  • Time Tracking: Live timer updates every second while tab is active
  • Tab Switching: Time pauses when tab is inactive
  • Background Requests: AJAX/Fetch don't trigger popups or visits
  • Same-Site Navigation: Navigating within same domain doesn't count as new visit
